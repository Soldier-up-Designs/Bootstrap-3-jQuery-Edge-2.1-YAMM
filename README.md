<b>Installing globally:</b>

&nbsp;&nbsp;<em>- Installation via npm. If you don't have npm yet:</em>

<code>$ curl https://npmjs.org/install.sh | sh</code>

&nbsp;Once you have npm:

<code>$ npm install http-server -g</code>

<em>This will install http-server globally so that it may be run from the command line.</em>

&nbsp;&nbsp;Usage:

<code>$ http-server [path] [options]</code>

<code>[path]</code> defaults to <code>./public</code> if the folder exists, and <code>./</code> otherwise.
